# 🧠 Machine Learning – Previsão de Intenção de Compra

## 📌 Sobre o projeto

Este projeto tem como objetivo prever a intenção de compra de usuários em um site de e-commerce, utilizando técnicas de machine learning aplicadas ao dataset **Online Shoppers Purchasing Intention**.

Foram utilizados diferentes modelos, avaliados e comparados com métricas apropriadas, além da aplicação de técnicas de interpretabilidade para entender o comportamento das previsões.

---

## 🚀 Abordagem

* Pré-processamento e tratamento de dados
* Codificação de variáveis categóricas
* Separação entre variáveis preditoras e variável alvo
* Validação cruzada (cross-validation)

---

## 🤖 Modelos utilizados

* Decision Tree
* Random Forest
* SVM (kernel RBF)
* XGBoost
* LightGBM
* CatBoost
* Regressão Logística
* KNN

---

## 📊 Avaliação

* Métrica principal: F1-score
* Comparação entre diferentes modelos
* Análise de overfitting com curvas de aprendizado

---

## 🔍 Interpretabilidade

* Uso de SHAP para análise global e local
* Comparação com LIME
* Identificação das variáveis mais relevantes para o modelo

---

## 🛠️ Tecnologias utilizadas

* Python
* Scikit-learn
* Pandas, NumPy
* Matplotlib, Seaborn
* XGBoost, LightGBM, CatBoost

---

## 🎯 Objetivo

Aplicar técnicas de machine learning para resolver um problema real de classificação, comparando diferentes abordagens e utilizando métodos de interpretabilidade para análise dos resultados.

---

## 📊 Resultados

* Comparação entre múltiplos modelos de classificação
* Identificação do modelo com melhor desempenho
* Análise interpretável das previsões
